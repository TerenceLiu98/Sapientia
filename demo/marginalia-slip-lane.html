Demo · PDF gutter notes + minimap rail

读者一眼能读到边距，扫一眼能看到全篇。

不再把 note 单独摆进独立第二列，而是让它们贴进 PDF preview 右侧的 gutter。当前视口附近的笔记像页边批注一样悬在页面旁边， 最右边一条定长 rail 继续承担全篇 minimap 和远距离跳转。

三层信息密度

贴页 (embed): 视口附近的笔记住进 PDF gutter，仍像 margin note。

扫篇 (scan): 远处的笔记压成 rail 上的有色 dot，颜色编码来源。

写入 (edit): 点 gutter slip 就地展开成编辑器，连接线仍指回锚点。

Reading systems should let commentary accumulate beside the sentence, not above it.

Scholarly notes become more alive when they remain visibly attached to the line that provoked them. A conventional sidebar often treats a note as a database object, while marginalia behaves like a situated response: an afterthought, a challenge, a synthesis, or a whispered cross-reference.

The geometry matters. When the reader can still sense page position, the note inherits the memory of the source passage, because the thought is recalled together with the visual neighborhood in which it was written.

A successful margin does not overwhelm the document. Instead it offers a light structure for response: one living note in focus, several quieter slips nearby, and just enough metadata to re-enter context without turning the act of reading into project management.

The page becomes a duet: the document supplies voice, the margin supplies harmony. Neither overwrites the other; instead, they read together as a single layered surface.

PDF gutter（页侧批注）

什么时候出现: 当前 PDF 视口及上下半页的 anchors。其它页面的 note 不在 gutter 出现。

折叠态: source tag + 正文前 1-3 行。读者扫一眼即可读到笔记本身。

Source tag 命名: highlight / underline 用 highlight n p. P blk. B（论文内序号 + 页码 + 所在 block），block-anchored 用 block N。一条 tag 自带全部坐标信息。

展开态: 折叠 slip 不弹独立浮层。它的右边仍贴 rail，向左生长进 PDF 留白区，宽度从 ~210px 涨到 ~520px，露出完整编辑器 chrome（jump / delete / close / 保存状态 / 左下角 resize 把手）。其它 slip 自然下移。

边缘 ghost: 视口外一点的 note 半透明出现，暗示"再滚一点就能看到"。

Rail（scrollbar + minimap）

是 scrollbar: 一条圆角凹槽 + 一颗 thumb。Thumb 的位置和高度 = 当前 PDF 视口在整篇里的位置和占比。拖 thumb 就是滚 PDF。

也是 minimap: 同一条凹槽上叠着每条 note 的 dot——位置 = anchor 在整篇里的位置, 颜色 = 来源 ( / / / · 多 cite = 饼图)。

Thumb 内的 dot: 全色全大小, 对应 gutter 里的 excerpt slip。

Thumb 外的 dot: 缩小、淡化。点击 = PDF 跳到那一页, gutter 重新组装。

Active dot: 当前展开的 note, 一根连接线指向 slip。

设计原则

读到 / 看到 / 写入三个动词三个图层, 信息密度逐级降低。

就地展开优于浮层: editor 嵌在 slip 里, 上下文不消失。

不抢 PDF 的位置: gutter 和 rail 加起来 ~280px, PDF 仍是主角。

Cite ≠ New note: bbox toolbar 上是两个图标, gutter 不参与创建 — 创建仅来自 PDF 的 highlight/block。
